trackdlo_perception

Real-time tracking of Deformable Linear Objects using ROS2 and Intel RealSense RGB-D cameras

ROS2 Humble / Jazzy BSD-3-Clause CUDA Optional
GitHub Repository Releases

Features

CPD-LLE Algorithm

Coherent Point Drift with Locally Linear Embedding for robust, real-time deformable object tracking with occlusion handling.

Pluggable Segmentation

Swap segmentation backends (HSV, YOLO, DeepLab) by subclassing SegmentationNodeBase. New methods added without touching core code.

4-Panel Preview

Live composite view showing camera feed, segmentation mask, overlay, and tracking results simultaneously.

GPU Acceleration

Optional CUDA support for point cloud generation. Auto-detected at build time — falls back to CPU seamlessly.

Docker Ready

Single command launch with automatic NVIDIA GPU detection. No manual dependency installation needed.

RealSense Support

Works with Intel RealSense D415, D435, D455 series RGB-D cameras out of the box.

colcon build --cmake-args -DCMAKE_BUILD_TYPE=Release
source install/setup.bash
# Build and run
cd docker/
bash build.sh
xhost +local:docker
./run.sh              # HSV segmentation
./run.sh hsv_tuner    # HSV with GUI tuner

Processing Pipeline

[RealSense D435/D415] │ ├── /camera/color/image_raw ├── /camera/aligned_depth_to_color/image_raw │ ▼ [Segmentation] ← HSV / YOLO / DeepLab (swappable) │ ▼ /trackdlo/segmentation_mask │ [trackdlo_node (CPD-LLE)] │ ├── /trackdlo/results_pc → tracked DLO node positions ├── /trackdlo/results_img → visualization overlay │ ▼ [composite_view] ← 4-panel preview window
